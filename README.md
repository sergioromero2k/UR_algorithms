# Algorithms & Data Structures

This repository contains implementations of classic algorithms organized by category.
It covers **greedy algorithms**, **graph problems**, and combinations of both — including
shortest path algorithms, minimum spanning trees, topological sorting, and more.

Each problem is sorted by the main technique used to solve it, making it easy to
navigate and use as a reference for studying or competitive programming.
